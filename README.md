# Telegram bot: daily EiBi stations

Бот каждый день:

1. в `23:55 UTC` обновляет локальную SQLite-базу данными из `eibi.txt`,
2. в `00:00 UTC` отправляет уведомление об обновлении.

## Возможности

- Просмотр станций по языку вещания (кнопки с языками)
- Выбор конкретного языка и получение всех станций, вещающих на нём
- Поиск станций по частоте
- Показ текущего времени UTC
- Показ всех частот и времён вещания для каждой станции в удобном списке
- Ручное обновление базы данных
- Автоматическое ежедневное обновление и уведомление

## Источники

- Список: http://www.eibispace.de/dx/eibi.txt
- Описание формата: http://www.eibispace.de/dx/README.TXT

## Установка

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## Настройка

1. Скопируйте `.env.example` в `.env`.
2. Заполните:
   - `TELEGRAM_BOT_TOKEN`
   - `TELEGRAM_CHAT_ID`
   - `EIBI_DB_PATH` (опционально, путь к SQLite базе; по умолчанию `data/eibi.sqlite3`)

Как получить `chat_id`:

- Напишите боту в Telegram.
- Откройте:
  `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`
- Возьмите значение `chat.id`.

## Запуск

```powershell
python bot.py
```

После запуска бот будет ждать и:
- обновлять базу ежедневно в `23:55 UTC`,
- отправлять уведомление об обновлении в `00:00 UTC`.

## Команды

- `/start` — показать доступные команды.
- `/now` — выбрать язык и получить станции на сегодня.
- `/datetime` — показать текущие дату и время UTC.
- `/freq` — найти станции по частоте (интерактивный ввод: бот спросит частоту, вы введёте число).
- `/refresh` — обновить базу данных вручную.

При вводе `/` в поле ввода Telegram показывает подсказки с командами.

## Формат вывода

Станции отображаются с группировкой частот по времени вещания (от 00:00 и далее):

```
R (Russian):
  ★ AF Transport Davlenie (RUS):
    • 6685, 8847, 11360, 18030kHz 0000-2400
E (English):
  ★ BBC (G):
    • 1386kHz 1830-1900
    • 1386kHz 2000-2030
  ★ HCJB Voice of Andes (EQA):
    • 3995, 5920kHz 0400-0500
    • 3995, 5920kHz 2300-0100
    • 7365kHz 0000-2400
```

## Интерфейс языков

- Кнопки показывают языки с количеством станций (отображаются языки с 10+ станциями).
- Специальные коды (`-TS`, `-CW`, `-TY` и др.) показываются всегда.
- Используются человекочитаемые подписи (например, `R (Russian)`, `E (English)`), а если код неизвестен — показывается оригинальный код из EiBi.

## Важно про формат дней

В `eibi.txt` есть простые и специальные обозначения дней.
В боте поддержаны:

- ежедневные записи (без поля дней),
- числовой формат (`1234567`, где `1=Mo` ... `7=Su`),
- перечень дней (`Mo,Tu,We`),
- диапазон (`We-Mo`),
- первый N-й день недели в месяце (`1.Sa`, `2.Fr`),
- последний день недели в месяце (`Last7`, `LastFr`),
- "чередующиеся" дни (`altFr` — учитывается как пятница),
- формат с ограничением по дате месяца (`MF-15`),
- формат конкретной даты (`15Sep`).

Строгий режим: редкие метки без однозначной календарной логики (`Ram`, `Haj`, `irr`, `test`, `tent`, `alt` и т.д.) не включаются автоматически.
